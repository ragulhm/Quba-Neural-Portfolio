# 📗 Astral Architect IO: Technical Reference & Project Guide

## 1. Executive Project Review
**Astral Architect IO** has evolved from a dynamic portfolio into an **Intelligent Monolith**. By integrating **Django 5.0** with **Google Gemini 2.0**, the project now features a real-time "Knowledge Matrix" that aggregates global activity from GitHub and Medium to provide an interactive, AI-driven experience for visitors and recruiters.

### Key Strengths:
- **Intelligent Assistant**: Features "Quba", a custom-tuned AI concierge that leverages a localized knowledge base of ~30+ repositories to answer technical queries.
- **Deep Integration**: Automatically fetches and indexes real-time GitHub pulses and Medium articles, ensuring the portfolio is always current without manual updates.
- **Node-Optional Architecture**: Uses a standalone Tailwind CLI workflow, eliminating the need for a persistent Node.js runtime while maintaining high-end UI capabilities.
- **Advanced Aesthetics**: Implements a "Glassmorphism" design system with smooth GSAP animations and a custom WebGL 3D engine.

## 2. In-Depth Concepts

### A. The "Quba" Intelligence Core
The AI assistant (Quba) is the technological centerpiece of version 2.0.
- **Engine**: Powered by `gemini-2.5-flash-lite` via the `google-genai` SDK.
- **Knowledge RAG (Retrieval-Augmented Generation)**: Unlike generic LLMs, Quba uses a localized `github_knowledge.json` generated by `extract_knowledge.py`. This ensures it can discuss specific technical implementation details across Ragul's entire codebase.
- **Metrics-Driven**: The system prompt is engineered to provide precise, data-backed insights (Stars, Forks, Tech Stacks) to demonstrate technical impact.

### B. The "Astral System" WebGL Engine
The 3D visualization in `static/js/three_scene.js` is a real-time WebGL engine built on **Three.js**.
- **Geometry Logic**: Uses an `IcosahedronGeometry` with `flatShading: true` for a faceted, crystalline look.
- **The Particle Nebula**: Instantiates `1,500 BufferGeometry` points with randomized spherical distribution.
- **Parallax Math**: mouse-tracking with dampened offsets creates a premium, responsive feel:
  ```javascript
  this.scene.position.x += (this.targetMouse.x * 0.5 - this.scene.position.x) * 0.05;
  ```

### C. Real-Time Data Pipelines
The portfolio acts as a living document through two primary pipelines:
- **GitHub Pulse**: `GitHubAPI` fetches recent commit messages and repository activity, displayed in a "Live Pulse" feed within the AI panel.
- **Medium Articles**: `MediumAPI` parses RSS feeds into JSON, allowing the portfolio to serve as a technical blog hub without hosting overhead.

## 3. Case Studies & Use Cases

### Case 1: The Automated Recruiter Experience
**Use Case**: A recruiter visits to assess backend skills.
- **Impact**: Instead of browsing repos, they ask Quba: "What is Ragul's most complex project?". Quba analyzes the Knowledge Matrix and recommends a specific repo with its impact metrics and tech stack.
- **Outcome**: A 10x faster assessment period for potential stakeholders.
